### Boas práticas: <br>
 Parametros sao nomeados por letras maisuclas unicas    <br>
**E** - Element<br>
**T** - type<br>
**V** - Value<br>
**K** - Key<br>
**N** - Number<br>

S, u V etc.. segunto terceiro e quantos tipos de uma lista parametros
